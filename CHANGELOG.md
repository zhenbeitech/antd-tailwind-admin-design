## [1.0.0] - 2026-04-13

### 新增
- 设计理念文档（DESIGN.md）
- Ant Design 6 + Tailwind CSS 4 样式系统
- CSS 变量桥接层（tokens/tokens.css）
- Tailwind CSS v4 配置（tokens/styles.css）
- 核心组件示例（Button、Table）
- 页面模板（列表页、详情页）
- 布局组件（PageContainer、ContentSurface）
- 语义化样式系统
- 双模式主题支持（浅色/深色）
- AI 使用指引
- 完整的项目配置（package.json、.npmrc、.gitignore）

### 文档
- 详细的设计规范文档
- 组件使用示例
- 页面模板代码
- 快速上手指南
- 贡献指南

### 技术特性
- 基于三层架构：AntD Token → CSS Variable → Tailwind Class
- 零硬编码色值，完全使用 Design Token
- 响应式设计支持
- 无障碍访问支持
- 性能优化建议

- **次版本号(x.1.x)**: 新功能添加，向后兼容
